# プロジェクト概要

**販売アシスト1号 / 母艦「お局様」**

---

## 🎯 プロジェクトの目的

**スマホ1台で完結する販売管理システム**

「スマホは20年前のスーパーコンピューターより高性能大容量なので、販売管理システムなんか簡単に出来る筈」という理念のもと、スマートフォン単体で企業の販売管理業務を完全に処理できるシステムを構築します。

---

## 💡 コアコンセプト

### 1. オフラインファースト

**ネットワーク不要で全業務完結**
- SQLiteによる完全なローカルデータ保存
- 見積・受注・売上・請求・在庫管理
- PDF生成・印刷
- すべての機能がオフラインで動作

### 2. スマホの性能を最大限活用

**20年前のスーパーコンピューター級の性能**
- 8コアCPU、8-16GBメモリ
- 128-512GBストレージ
- GPS、カメラ、各種センサー
- 5G/WiFi/Bluetooth通信

**実現していること**
- 数万件の伝票・商品・顧客データ管理
- リアルタイム集計・分析
- 高速検索・フィルタリング
- PDF生成・送信
- 写真・バーコードスキャン
- GPS位置記録

### 3. 汎用テンプレートシステム

**コードの重複を90%削減**
- `GenericListScreen<T>` - 150行で完全なリスト画面
- `DocumentCard` - 伝票カード表示
- `BaseDocument` - 共通伝票モデル

**実績**
- 見積入力: 150行
- 受注入力: 140行
- 売上入力: 150行
- 売上返品入力: 130行

### 4. 業種を問わない汎用性

**焼き芋屋から建設業まで対応**
- 業種プロファイル設定（計画中）
- カスタムフィールド機能（計画中）
- 業種別テンプレート（計画中）

### 5. 母艦「お局様」との連携（オプション）

**オンライン時の強化機能**
- LAN/Gmail経由での同期
- GPS位置ベース自動切り替え
- 7年間のデータ保存
- 複数端末の一元管理

## 🎨 システム構成

### 二層構造

```
┌─────────────────────────────────┐
│  販売アシスト1号（スマホアプリ）  │
│  ・オフライン完結                │
│  ・全業務機能                    │
│  ・SQLiteデータベース            │
└─────────────────────────────────┘
         ↕ 同期（オプション）
┌─────────────────────────────────┐
│  母艦「お局様」（サーバー）       │
│  ・データ集約・バックアップ      │
│  ・複数端末管理                  │
│  ・7年保存                       │
└─────────────────────────────────┘
```

### 動作モード

| モード | 説明 | 主な機能 |
|--------|------|----------|
| **スタンドアローン** | 端末単体で完結 | 見積・受注・売上・在庫管理 |
| **オンライン連携** | 母艦と接続 | データ同期・バックアップ・一元管理 |

## 👥 対象ユーザー

### 主要ターゲット

1. **小規模事業者**
   - 焼き芋屋、たいやき屋などの移動販売
   - 個人商店、小売店

2. **専門業者**
   - ホームページ制作業
   - 電気工事業
   - 建設建築業

3. **配達・通販業**
   - 灯油配達
   - 米の通販
   - 定期配送業

### ユーザーのニーズ

✅ **シンプル**: 複雑な操作は不要  
✅ **安価**: 高額なシステムは導入できない  
✅ **オフライン**: ネット環境がない場所でも使える  
✅ **柔軟**: 業種固有の要件に対応  
✅ **安全**: データが失われない

**業種を問わない**
