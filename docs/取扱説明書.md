# 販売アシスト1号 取扱説明書

---

## 1. アプリ概要

**販売アシスト1号**は、オフラインで完結する販売管理アプリケーションです。見積・納品・請求・売上・在庫管理などの業務をスマートフォン1台で行うことができます。

### 主な機能

- 見積書・納品書・請求書の作成・発行
- 売上入力・レジ機能
- 商品マスター・得意先マスター管理
- 在庫管理・棚卸
- 売上分析・日報
- PDF帳票の出力・共有
- Google Driveバックアップ（オプション）

### 動作環境

- **対応OS**: Android 7.0以上
- **必要な権限**:
  - ストレージ（PDF保存・データベースアクセス）
  - 連絡先（得意先情報の取り込み）
  - 位置情報（母艦検出機能、オプション）
  - インターネット（Google連携・同期、オプション）

## 2. インストール方法

### APKからのインストール

1. 配布されたAPKファイルを端末にダウンロード
2. 「設定」→「セキュリティ」で「不明なソースからのアプリ」を許可
3. APKファイルをタップしてインストール
4. インストール完了後、アプリを起動

### 初期設定

初回起動時に以下の設定を行います：

1. **会社情報の登録**
   - 会社名・住所・電話番号
   - 登録番号（適格請求書発行事業者の場合）
   - 銀行口座情報

2. **税率設定**
   - デフォルト税率（通常10%）
   - 軽減税率（8%）の使用有無

3. **印影設定**
   - 角印画像の登録（オプション）

## 3. 基本操作

### 3.1 ホーム画面

アプリ起動時に表示されるメイン画面です。以下の機能にアクセスできます：

- **ダッシュボード**: 売上概要・最近の伝票
- **マスタ管理**: 商品・得意先・仕入先・倉庫・担当者
- **販売管理**: 見積・受注・売上・請求書
- **仕入管理**: 発注・仕入・支払予定
- **在庫管理**: 在庫照会・移動・棚卸
- **集計分析**: 日報・売上推移・粗利分析
- **設定**: 各種設定・Google連携・バックアップ

### 3.2 画面遷移

- 各メニューをタップすると該当画面に遷移
- 左上の「←」ボタンで前の画面に戻る
- ホーム画面はメニューボタンからアクセス

### 3.3 データの保存

- 各画面で「保存」ボタンをタップするとデータが保存されます
- 一部の画面では自動保存されます
- 保存済みのデータは編集・削除が可能です

## 4. 主要機能の使い方

### 4.1 見積書の作成

1. 「販売管理」→「見積入力」をタップ
2. 得意先を選択
3. 商品を追加（「商品を追加」ボタン）
4. 数量・単価を入力
5. 備考が必要な場合は入力
6. 「保存」ボタンをタップ
7. PDFプレビューで確認後、「共有」で送信

### 4.2 売上入力（レジ機能）

1. 「販売管理」→「売上入力」をタップ
2. 得意先を選択（または現金売上の場合はスキップ）
3. 商品を追加
4. 数量を入力
5. 合計金額を確認
6. 「保存」ボタンをタップ

### 4.3 請求書の発行

1. 「販売管理」→「請求書発行」をタップ
2. 請求書に含める売上を選択
3. 請求書の件名・発行日を入力
4. 「保存」ボタンをタップ
5. PDFで出力・共有

### 4.4 商品マスターの登録

1. 「マスタ管理」→「商品マスター」をタップ
2. 「＋」ボタンで新規追加
3. 商品名・コード・単価を入力
4. 在庫管理が必要な場合は初期在庫を入力
5. 「保存」ボタンをタップ

### 4.5 得意先マスターの登録

1. 「マスタ管理」→「得意先マスター」をタップ
2. 「＋」ボタンで新規追加
3. 得意先名・住所・電話番号を入力
4. 連絡先から取り込むことも可能
5. 「保存」ボタンをタップ

### 4.6 在庫管理

1. 「在庫管理」→「在庫照会」をタップ
2. 商品別の在庫数を確認
3. 在庫移動・棚卸入力から在庫を調整

### 4.7 売上分析

1. 「集計分析」→「売上日報」をタップ
2. 日付範囲を指定
3. 売上合計・商品別売上を確認
4. 「得意先別売上推移」「商品別粗利分析」で詳細分析

## 5. PDF帳票の出力・共有

### 5.1 PDF出力

- 見積書・納品書・請求書などからPDFを出力可能
- プレビュー画面で内容を確認
- 「保存」で端末に保存
- 「共有」でメール・メッセージアプリ等で送信

### 5.2 メール送信

- 共有機能からメールアプリを選択
- BCC設定（設定画面で指定したアドレス）が自動追加
- 宛先・件名・本文を編集して送信

---

## 6. Google連携・バックアップ（オプション）

### 6.1 Googleアカウント連携

1. 「設定」→「Googleアカウント連携」をタップ
2. 連携するGoogleアカウントを選択
3. 権限を許可
4. 連携完了

### 6.2 Google Driveバックアップ

1. 「設定」→「バックアップ」をタップ
2. 「今すぐバックアップ」で手動バックアップ
3. 自動バックアップを有効にすると定期的にバックアップ
4. バックアップから復元も可能

### 6.3 母艦システム連携（オプション）

- お局様サーバーとのデータ同期
- GPS位置ベースの自動検出
- チャット機能

## 7. データの取り扱い

### 7.1 データ保存場所

- 全データは端末内のSQLiteデータベースに保存
- オフラインで動作可能
- インターネット接続は不要（バックアップ・同期時を除く）

### 7.2 プライバシー

- 位置情報は母艦検出機能のみに使用
- 連絡先情報は得意先登録時にのみ使用
- データは外部サーバーに送信されません（バックアップ・同期時を除く）
- Google連携時はGoogleのプライバシーポリシーに従います

### 7.3 データの削除

- アプリをアンインストールするとデータは削除されます
- バックアップがない場合、データは復元できません
- 定期的なバックアップを推奨します

## 8. トラブルシューティング

### 8.1 アプリが起動しない

- 端末を再起動
- アプリを再インストール
- OSバージョンが対応しているか確認

### 8.2 データが保存されない

- ストレージ権限が許可されているか確認
- 端末の空き容量を確認
- アプリを再起動

### 8.3 PDFが出力されない

- ストレージ権限が許可されているか確認
- ダウンロードフォルダの権限を確認
- アプリを再起動

### 8.4 Google連携ができない

- インターネット接続を確認
- Googleアカウントが正しく設定されているか確認
- Google Play Servicesが最新版か確認

### 8.5 バックアップが失敗する

- インターネット接続を確認
- Google Driveの空き容量を確認
- Googleアカウントの権限を確認

## 11. よくある質問（FAQ）

**Q: オフラインで使用できますか？**
A: はい、インターネット接続なしで全機能を使用できます。バックアップ・同期機能のみオンラインが必要です。

**Q: データはどこに保存されますか？**
A: 端末内のSQLiteデータベースに保存されます。外部サーバーには送信されません（バックアップ・同期時を除く）。

**Q: 複数端末でデータを共有できますか？**
A: 母艦システム連携機能を使用することで、複数端末でのデータ共有が可能です。

**Q: 印刷はできますか？**
A: PDF出力後、プリンタ対応アプリから印刷可能です。

**Q: 税率設定は変更できますか？**
A: はい、設定画面からデフォルト税率を変更できます。また、明細ごとに税率を指定することも可能です。

**Q: 適格請求書に対応していますか？**
A: はい、適格請求書発行事業者の登録番号を設定することで対応しています。

## 12. 用語集

- **見積書**: 取引前に発行する見積もり書類
- **納品書**: 商品納品時に発行する書類
- **請求書**: 代金請求時に発行する書類
- **売上**: 商品・サービスの販売実績
- **レジ**: 店頭での販売入力機能
- **マスター**: 基本データ（商品・得意先など）
- **在庫**: 商品の在庫数
- **棚卸**: 実際の在庫数を確認する作業
- **粗利**: 売上から原価を引いた利益
- **母艦**: データ同期・監視を行うサーバーシステム
